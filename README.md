# Robots to Mars

### Run application

```
npm run start
```

To run in watch mode

```
npm run start:dev
```

### Run test

```
npm run test
```

Run in watch mode

```
npm run test:dev
```

To see code coverage

```
npm run test:cov
```

### Run MongoDB

```
docker-compose up -d
```

## Technologies

- [Express.js](https://expressjs.com/)
- [Node.js](https://nodejs.org/es/)
- [Jest](https://jestjs.io/)
